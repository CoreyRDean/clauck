# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project, you agree to abide by its terms.

## Reporting

Report issues to the maintainer via the contact methods listed in the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) or by opening a private security advisory on GitHub.
